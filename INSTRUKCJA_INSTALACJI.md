# 📱 Jak zbudować i zainstalować APK — krok po kroku

## Krok 1 — Załóż darmowe konto GitHub
1. Wejdź na https://github.com
2. Kliknij **Sign up** i załóż konto (wystarczy e-mail + hasło)

## Krok 2 — Wrzuć projekt na GitHub
1. Zaloguj się na https://github.com
2. Kliknij zielony przycisk **New** (lewy górny róg)
3. Wpisz nazwę repo np. `dual-camera` → kliknij **Create repository**
4. Na następnej stronie kliknij link **uploading an existing file**
5. Przeciągnij na stronę **wszystkie pliki z wypakowanego ZIPa**
6. Kliknij **Commit changes**

## Krok 3 — Poczekaj na build (~10 minut)
1. Po wrzuceniu plików automatycznie uruchomi się build
2. Kliknij zakładkę **Actions** (górne menu repo)
3. Zobaczysz wpis **Build APK** — czeka, aż pojawi się zielony ✅
4. Kliknij na ten wpis → przewiń na dół → kliknij **dual-camera-app**
5. Pobierze się plik ZIP z APK w środku

## Krok 4 — Zainstaluj APK na S25 Ultra
1. Wyślij plik `app-release.apk` na telefon (np. e-mailem lub przez Google Drive)
2. Otwórz plik APK na telefonie
3. Jeśli telefon pyta o zgodę — wejdź w **Ustawienia → Zainstaluj nieznane aplikacje** i zezwól
4. Zainstaluj i gotowe! 🎉
